Kommentarer och titel i HTML

 

<!--...--> används för att lägga in osynliga kommentarer i HTML-dokument.
Kommentarstaggar kan även användas om man vill att webbläsare inte skall tolka HTML-kod.

 

<!--Denna HTML-skola är gjord av ........................-->

<!-- <h1><center>Min hemsida</center></h1> -->


<!DOCTYPE HTML PUBLIC "-//W3C// DTD HTML 4.0// EN">
<!doctype...>
  Ovanstående tagg är den enda tagg som kan stå före html-taggen. Här kan man t. ex.
ange önskad version av html.

 

Titel

 

<title>...</title> Identifierar innehållet i dokumentet. Det som skrivs mellan dessa taggar visas
högst upp till vänster i webbläsarens namnlist.

 

<title> Min hemsida. Här kan du läsa om mina tatueringar</title> 

 

Sidhuvud

 

<head>...</head> Dessa taggar utgör en webbsidas sidhuvud och innehåller ofta
script av olika slag, titeln samt information för sökmotorer (META-taggar).

<head> script, metataggar och titel</head>


<link> Används för att skapa referens till annat dokument och skrivs i sidhuvudet.
<link href="style.css" rel="stylesheet" type="text/css">


Metataggar (skrivs i sidhuvudet, head)

<meta name="generator" content="notepad">
Anger vilket program du använt för att skapa html-filen.

<meta name="keywords" content="sökord, sökord, sökord">
Sökord som registreras hos sökmotorer. Avskilj sökorden med kommateceken.

<meta name="description" content="beskrivning av sidan">
Beskrivning av sidans innehåll i söktjänsten.

<meta name="author" content="ditt namn">
Visar författarens namn i html-koden.

<meta http-equiv="refresh" content="5">
Sidan uppdateras efter 5 sekunder.

<meta http-equiv=refresh" content="10; url=http://www.domän.se">
Sidan uppdateras med en annan sida efter 10 sekunder.

<meta http-equiv="expires" content="fri, 13 Jan 2002 24:00:00 GMT>
Laddar in sidan på nytt från servern efter angivet datum.

Bodytaggar

 

<body>...</body> Dessa taggar skall omsluta allt visuellt i HTML-dokumentet.

 

Attribut:

 

background: Används för att infoga bakgrundsbild på webbsidan.

bgcolor: Används för att ge webbsidan bakgrundsfärg. Man kan antingen ange hexadicimal färgkod eller direkt namnge färgen. Lägg märke till stavningen på color.

 

<body bgcolor="#330066">

<body bgcolor="brown">

 

link, alink, vlink: Används för att bestämma vilka färger länkar ska ha innan.
Link är den färg länken har om man inte har besökt adressen den pekar på.
Alink är den färg länken har just i klickögonblicket.
Vlink är den färg länken har efter att man har besökt adressen som länken pekar på.

Bilder

<area>

<area> Används för att definiera klickbara ytor på en image map.

 

Attribut:

 

href: Används för att ange adressen till källan man vill nå.

 

shape: Används för att definera vilken form den klickbara ytan på imagemap skall ha.

 

<area href="mig.html" shape=rect>

<area href="mig.html"shape=circle>

<area href="mig.html" shape=poly>

 

coords: Används för att ange kordinaterna för den klickbara ytan i en imagemap.

 

Om man har rect som form anger man x och y kordinaterna för övre vänstra hörnet
och nedre högra hörnet.

<area href="mig.html" shape=rect coords="0, 0, 50, 50">

Om man använder sig av circle anger man x och y kordinater för centerpunkten
samt omkretsen för cirkeln.

<area href="mig.htm"l shape=circle coords="25, 25, 158">

Om man använder sig av poly anger man x och y för varje brytningspunkt i polygonen.

<area href="mig.htm"l href="mig.html" shape=circle coords="25, 12, 50, 56, 123, 34"
och så vidare........>

 

target: Används för att styra var källan till länken skall visas.

<area href="mig.html" target=_parent>Läs om mig</a>

Frames

 

<frame> Definierar innehållet i en ram.

 

Attribut:

 

src: Värdet man ger src är sökvägen till det dokument man vill ha i ramen.

 

<frame src="minmeny.html">

 

name: Värdet man tilldelar name blir ramens namn.

 

<frame src="minmeny.html" name=meny>

 

frameborder/border: Kan anta 0 och 1 som värde.
Om man tilldelar attributen 0 visas inga ramar.

 

<frame src="minmeny.html" name=men" frameborder=0 border=0>

 

marginwidth/marginheight: Dessa attribut bestämmer hur stor marginal man vill ha

mellan en rams innehåll och dess kanter.

 

<frame src="minmeny.html" name=meny marginwidth=0 marginheight=10>

 

noresize: Detta attribut gör det omöjligt för besökaren av din webbsida att
ändra ramarnas storlekar.

 

<frame src="minmeny.html" name=meny noresize>

 

scrolling: Bestämmer om scroll-lister skall synas eller ej. Värdena som man kan sätta
till scrolling är auto, yes och no.

 

<frame src="minmeny.html" name=meny noresize scrolling=no>

Formulärtaggar

 

<form>...</form> Används för att skapa ett formulär som innehåller kontroller
som användaren kan klicka på eller skriva i.

 

Attribut:

 

action: Används för att ange adress till bakomliggande script som
bearbetar informationen som användaren skickar iväg.

 

<form action="mittscript.cgi">

 

method: Anger på vilket sätt data skickas från formuläret till det bakomliggande scriptet.
Detta värde kan antingen vara get eller post.

 

<form action="mittscript.cgi" method="post">

<form action="mittscript.cgi" method="get">

 
enctype: Anger vilken mediatyp sm skall användas för att koda informationen innan
den skickas till webbläsaren och manuset.
<form action="mittscript.cgi method=post enctype="text/plain">

Om du vill att informationen skall skickas till din e-post skriver du:

<form method="post" action= "mailto:flitiga@lisa.se"
enctype="text/plain">



Formulärkomponenter

<input> Används för att skapa diverse formulärkomponenter.

 

Attribut:

 

type: Detta attribut bestämmer vilka slags formulärskomponenter
man vill använda och kan anta:
text, submit, reset, radio, checkbox, file, hidden, image, button och password.

 

<input type=text>

 

size: Med size kan man bestämma storlek på textfält.

 

<input type=text size=14>

 

name: Med name tilldelar du formulärkomponenten ett namn.
Detta är ofta nödvändigt för att script skall veta från vilka formulärkomponeneter
den skall hämta information ifrån.

 

<input type=text size=14 namn="emailadress">

 

value: Används för att tilldela formulärkomponenterna värden.

 

<input type=text size=14 namn=emailadress value="ellington@rocketmail.com">

 

Om man använder radio eller checkbox så kan man lägga till ytterligare ett attribut som heter checked. Lägger man in det attributet blir formulärkomponenten automatiskt förkryssad.

 

<input type=radio checked>

rowspan/colspan: Bestämmer hur många kolumner eller rader en cell skall spänna över.

 

<table>

<tr>

<td colspan=2>

Pizza

</td>

</tr>

<tr>

<td>Quattro</td>

<td>Vesuvio</td>

</tr>

</table>

 

width/height: Bestämmer bredd och höjd på cellen.

 

<td colspan=2 width=200 height=20>

 

align: Bestämmer hur cellens innehåll skall placeras i horisontellt led.
Attributet kan anta left, right och center som värden.

 

<td colspan=2 width=200 height=20 align=center>

 

valign: Bestämmer hur cellens innehåll skall placeras i vertikalt led.
Attributet kan anta top, bottom och middle som värden.

 

<td colspan=2 width=200 height=20 align=center valign=middle>

 

bgcolor/background: Med dessa attribut kan du bestämma bakgrundsfärg eller
bakgrundsbild på tabellceller.

 

<td background="janne.jpg">

nowrap: Detta attribut förhindrar webbläsaren att göra radbrytingar på text i tabellceller.

 

<td background="janne.jpg" nowrap>

Inbäddad bakgrundsmusik

<embed src="bakgrundsmusik"> Spelar ingen roll vilken webbläsare som används,
måste däremot ha ett tilläggsprogram för ljudspelare.

Attribut:

autostart="true/false"
true spelar musiken automatiskt och false låter surfaren starta musiken.

loop="true/1"
true spelar upp musiken gång på gång. 1 anger hur många gånger låten skall spelas.

hidden="true/false"
true gömmer ljudspelaren, false visar ljudspelaren.

control="console/small console"

console anger typ av ljudspelare, small console ger en mindre variant.

width/height

anger ljudspelarens bredd respektive höjd i pixlar.

align="left/right/center/top/bottom/middle
align anger som vanligt placeringen i webbläsarfönstret.

volume=""-5000"
Anger ljudvolym, -10 000 till 0. 0= användarens ljudnivå

balans="10000"
Uppspelning i högtalare. Värden mellan -10000 och 10000. 0=lika

border="1/0"
Storlek på ram/döljer ram
