Kapitel 7 - Listor

Med listor kan man på ett snyggt sätt punktvis presentera information i HTML. Listar upp elementen gör du med taggen <LI> , som betyder list item. Det finns lite olika sorters listor man kan göra, nedan följer några exempel.

 

Oordnade listor
UL står för Unordered List (oordnade listor). Det är viktigt att avsluta för annars tror webbläsaren att allt efterkommande kod tillhör listan.

Lista

Koder

Favoritlag

  • Timrå

  • Modo

  • Björklöven
<UL>
<LI>Timrå
<LI>Modo
<LI>Björklöven
</UL>

 

UL-taggen har TYPE som attribut. TYPE bestämmer vilken sorts punkter man ska ha, alltså bestämmer utseendet på punkterna, väljer man inget blir värdet automatiskt disc. Som värde till TYPE kan man sätta:

( Dessa attribut funkar inte i vissa äldre webbläsare, Microsoft Explorer 3.0 t.ex).

Ordnade listor
Om elementen i en lista har en bestämd ordning, kan det vara en fördel kunna räkna upp dem. Man gör då på samma sätt som en oordnad lista, men använder istället taggen <OL> , som står för ordered list.

Lista

Koder

Favoritlag

  1. Timrå
  2. Modo
  3. Björklöven
<OL>
<LI>Timrå
<LI>Modo
<LI>Björklöven
</OL>


Precis som i oordnade listor kan man använda sig av attributet TYPE för att bestämma utseende på sin numrerade lista.

TYPE=1

Vanliga arabiska siffror: 1, 2 osv.

TYPE=a

Gemena bokstäver: a,b osv.

TYPE=A

Versaler: A, B osv.

TYPE=i

Små romerska siffror: i, ii osv.

TYPE =I

Stora romerska siffror: I, II osv.

Listor i listor
Ibland kan man i komplicerade sammanhang behöva använda listor i flera nivåer. Det är då vanligast att man har en oordnad lista inuti en ordnad.

Lista

Koder

  1. Starta Notepad
  2. Skriv kod
    • Spara som
    • Starta webbläsaren
  3. Uppdatera
<OL>
<LI>Starta Notepad
<LI>Skriv kod
<UL>
<LI> Spara som
<LI>Starta
webbläsaren
</UL>
<LI>Uppdatera
</OL>
